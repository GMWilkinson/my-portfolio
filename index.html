This project is made using Node.js, MongoDB, Express, ReactJS, JavaScript and SCSS. I decided to make an app where you can create your own adventure game-book and read all the books that have been created by other users. The most difficult aspect of this project for me was trying to work out how the creator of a book would link the pages together through choices at the bottom of the page. At present the app does what it is supposed to but is not as user friendly as I would like and the styling is not finished. I am most proud of the fact that by changing states rather than changing urls for the page changes, the reader of the created books can’t cheat and go back a page as it takes you back to the start of the book . I still have some bugs to fix. I am currently making a second version of this project that I can put more time into, make it look much better and fix all of the bugs.

This is the first project that I've made as part of a team. We used Node.js, MongoDB, Express, AngularJS, JavaScript, SCSS and HTML. We decided to make an event planner app where the user can add, join and comment on events. We used OpenStreetMap and Nominatem so that when the user adds an event a pin will be put on the map at the event’s location on the show page. This project was made mobile-first. In the future this project would be made much more useful if you could add friends and only the friends could see your events and join them.
